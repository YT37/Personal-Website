# Personal Website

A highly interactive, cyberpunk-themed personal portfolio website built with Vite, TypeScript, and Tailwind CSS. This project features a unique aesthetic with glitch effects, dynamic theming, and interactive components.

## 🚀 Tech Stack

- **Framework:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React & React Icons

## ✨ Features

- **Cyberpunk/Hacker Aesthetic:** Custom glitch effects, matrix rain, hacker text scramble animations, and terminal-style typography.
- **Dynamic Theming:** Real-time theme switcher with presets (Matrix, Cyberpunk, Synthwave, etc.).
- **Interactive Terminal:** A fully functional terminal simulator with commands like `help`, `neofetch`, `ls`, `cd`, `cat`, and `echo` to explore portfolio info.
- **Custom Cursor & Spotlight:** Animated bracket-style cursor with a radial light effect that follows mouse movement.
- **Magnetic Buttons:** Buttons that magnetically attract toward the cursor on hover.
- **Interactive Mascot:** A floating mascot that reacts to user interaction and page scroll, intelligently avoiding the footer.
- **Awards Radar:** Interactive radar visualization with clickable award blips that display achievement details.
- **Experience Timeline:** Work history displayed as an animated timeline with roles, descriptions, and key contributions.
- **Skills Carousel:** Horizontally scrolling skill rows with hover-pause animation.
- **Blog Section:** Grid of blog post cards with tags and links to external posts.
- **Contact Form:** Animated contact form powered by Web3Forms with success/error feedback.
- **Dev Tools Easter Egg:** Styled console warning message for curious developers.
- **Responsive Design:** Fully responsive layout that works across all device sizes.
